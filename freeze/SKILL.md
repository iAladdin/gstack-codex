# Freeze

1. Read `../references/principles.md`.
2. Read `../references/safety.md`.

## Rules

- Define the allowed edit scope up front.
- Do not edit outside that scope unless the user approves.
- If the requested task cannot be completed within the scope, stop and say so clearly.
